CD-TEXT Provider

Standard: Aktiviert - CD-TEXT ist ein Offline-Provider ohne externe API
Standard: 50 (Mittel) - MusicBrainz hat 100 (Hoch)
Empfohlung: 50 für beste Datenqualität, 150 für Offline-Bevorzugung
CD-TEXT Verfügbarkeit:
Nur ~10-20% aller Audio-CDs enthalten eingebettete CD-TEXT Metadaten.
Vorteile: Offline-fähig, sofortiges Auslesen (< 1s), 100% genau
Nachteile: Keine Cover Art, oft nur Album-Daten ohne Track-Titel
Benötigte Tools:
cd-info (libcdio-utils), icedax (cdrkit), oder cdda2wav (cdrtools)
Installation: sudo apt install libcdio-utils icedax
